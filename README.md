# Opportunity Finder

A starter Next.js app for a sales-facing geographic opportunity finder.

## What this starter includes
- Geography search UI
- Server-side search route
- Mock structured results
- CSV export route
- Supabase schema for production use
- TypeScript types and scoring scaffold

## What still need to be built
- Authentication
- Live public-source ingestion
- Deduplication and scoring pipeline
- Saved searches and rep ownership
- Production export to XLSX
- Background refresh jobs

## Local setup
```bash
npm install
npm run dev
```

Open `http://localhost:3000`
